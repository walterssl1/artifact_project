Quick Check

Before troubleshooting, verify:

  • Battery is charged or plugged in via USB-C
  • CD player is powered on
  • CD inserted correctly (label side up)
  • Volume is turned up
  • Headphones fully plugged in
  • Hold switch is not enabled

Troubleshooting Guide

Having trouble with your CD player? Use the guide below to find common issues and step-by-step solutions.

CD Player Won't Turn On

Possible Cause: Battery is drained

  1. Connect USB-C charging cable
  2. Wait 10–15 minutes
  3. Press power button

Possible Cause: Hold switch enabled

  1. Locate Hold switch
  2. Slide switch to OFF position
  3. Try powering on again

Possible Cause: Device not powered correctly

  1. Press and hold power button for 2–3 seconds
  2. If still off, plug into power

CD Will Not Play

Possible Cause: CD inserted incorrectly

  1. Open lid using open switch
  2. Place CD label side up
  3. Close lid firmly

Possible Cause: CD dirty or scratched

  1. Remove CD
  2. Clean from center outward
  3. Try another CD

Possible Cause: Player paused

  1. Press Play/Pause button
  2. Check LCD playback time

CD Skips or Freezes

Possible Cause: Movement during playback

  1. Place player on flat surface
  2. Avoid shaking during use

Possible Cause: Dirty CD

  1. Clean CD surface
  2. Retry playback

Possible Cause: Low battery

  1. Charge device via USB-C
  2. Restart playback

CD Player Produces No Sound

Possible Cause: Volume turned down

  1. Press Volume Up button
  2. Check LCD volume display

Possible Cause: Headphones not connected

  1. Insert headphone jack fully
  2. Try another headset

Possible Cause: Playback paused

  1. Press Play/Pause
  2. Confirm timer is counting

CD Player Sound is Distorted

Possible Cause: Volume too high

  1. Lower volume
  2. Test at mid level

Possible Cause: Audio preset enabled

  1. Press Mode button
  2. Cycle presets
  3. Select Normal setting

Possible Cause: Damaged CD or headphones

  1. Try different CD
  2. Try different headphones

Buttons Not Responding

Possible Cause: Hold switch enabled

  1. Slide Hold switch OFF
  2. Try buttons again

Possible Cause: Device frozen

  1. Turn player off
  2. Wait 5 seconds
  3. Turn back on

Possible Cause: Low battery

  1. Charge using USB-C
  2. Retry controls

CD Player Won't Charge

Possible Cause: Loose cable

  1. Reconnect USB-C cable
  2. Check power source

Possible Cause: Faulty cable

  1. Try another USB-C cable
  2. Try different adapter

Possible Cause: Battery reset needed

  1. Plug into charger
  2. Leave connected 20 minutes
  3. Attempt power on
